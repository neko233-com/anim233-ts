Scroll Animations Intersection

Trigger animations based on viewport intersection and scroll direction. Link animations for scroll-driven playback.

Usage

const observer = Anim233.onScroll({
  target: '#my-element',
  container: '#scroll-container',
  threshold: 0.5,
  onEnter: (entry) => console.log('entered viewport'),
  onLeave: (entry) => console.log('left viewport'),
  onEnterForward: (entry) => console.log('scrolling down, entering'),
  onEnterBackward: (entry) => console.log('scrolling up, entering')
});

Options

OptionTypeDefaultDescription
containerstring | HTMLElementwindowScroll container
targetstring | HTMLElement''Element to observe
axis'x' | 'y''y'Scroll axis
thresholdnumber | [number, number] | string0Intersection threshold
repeatbooleanfalseRe-trigger on re-enter
debugbooleanfalseDebug mode

Callbacks

CallbackSignatureDescription
onEnter(entry) => voidElement enters viewport
onEnterForward(entry) => voidEntering while scrolling forward (down)
onEnterBackward(entry) => voidEntering while scrolling backward (up)
onLeave(entry) => voidElement leaves viewport
onLeaveForward(entry) => voidLeaving while scrolling forward
onLeaveBackward(entry) => voidLeaving while scrolling backward
onUpdate(entry) => voidContinuous update while in view

Methods

MethodDescription
.link(animations)Link AnimationInstance(s) for scroll-driven playback
.refresh()Refresh observer state
.revert()Revert observer

Properties

PropertyTypeDescription

const observer = Anim233.onScroll({
  target: '#box',
  threshold: 0.5,
  onEnter: () => {
    Anim233.animate({
      targets: '#box',
      x: [50, 0],
      opacity: [0, 1],
      duration: 800,
      ease: 'easeOutCubic'
    });
  }
});

Link Animations to Scroll

const anim = Anim233.animate({
  targets: '#box',
  x: 300,
  rotate: 360,
  duration: 1000,
  autoplay: false
});

const observer = Anim233.onScroll({
  target: '#box'
});
observer.link(anim);
Note: The ScrollObserver API provides the interface above. For scroll-driven animations in production, you can combine it with the linked animation pattern or use the native IntersectionObserver alongside Anim233's animate API.
