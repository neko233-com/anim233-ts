Anim233

A powerful, zero-dependency animation library combining DOTween-style chains, anime.js-style declarative API, WebGL shaders, and rich utilities — all in 12KB.

Get Started GitHub
Live Demo — Hero Animation

Features

▶

Tween Chains

DOTween-inspired sequential chaining: .to() .to() .delay() .play()

⚙

Declarative Animate

anime.js-style Anim233.animate() with full control over targets, easing, and callbacks.

⏱

Timeline

Sequence and orchestrate multiple animations with precise timing offsets.

🎨

40 Easings

Complete easing library including Back, Elastic, Bounce, and custom registration.

🔬

WebGL Shaders

Create real-time shader animations with hot-swap, uniforms, and auto-time.

✊

Draggable

Physics-based drag interactions with spring, snap, and container confinement.

🔎

Text Splitting

Split text into characters for scramble and reveal effects.

📈

Scroll Animations

Trigger animations based on scroll position and viewport intersection.

🔨

Math Utilities

lerp, damp, clamp, mapRange, stagger, and more.

Quick Start

Install via npm:

npm install anim233-ts

Or use CDN:

<script src="https://unpkg.com/anim233-ts@latest/dist/index.min.js"></script>

Animate in 2 lines:

Anim233.tween('#box')
  .to({ x: 200, rotate: 360 }, 800, 'easeOutCubic')
  .to({ scale: 1.5 }, 400, 'easeOutBack')
  .play();
