Draggable Interaction

Create physics-based drag interactions with spring physics, container confinement, and axis locking.

Usage

const drag = Anim233.draggable('#box', {
  container: '#area',
  x: true,
  y: true,
  onGrab: () => console.log('grabbed'),
  onDrag: () => console.log('dragging'),
  onRelease: () => console.log('released'),
  onSettle: () => console.log('settled')
});

Options

OptionTypeDescription
triggerstring | HTMLElementDrag trigger element (defaults to target)
containerstring | HTMLElementConfinement container
containerPaddingnumberPadding inside container
containerFrictionnumberFriction at container edges
releaseContainerFrictionnumberFriction on release
releaseMassnumberPhysics mass on release
releaseStiffnessnumberSpring stiffness
releaseDampingnumberSpring damping
velocityMultipliernumberVelocity multiplier
minVelocitynumberMin velocity threshold
maxVelocitynumberMax velocity cap
releaseEaseEasingName | EasingFunctionRelease easing function
dragSpeednumberDrag speed multiplier
dragThresholdnumberMin movement to start drag
cursorstringCustom cursor style
xboolean | objectEnable X axis (or configure snap/modifier/mapTo)
yboolean | objectEnable Y axis (or configure snap/modifier/mapTo)

Callbacks

CallbackSignatureDescription
onGrab(e) => voidWhen drag starts
onDrag(e) => voidDuring drag movement
onUpdate(e) => voidPosition update
onRelease(e) => voidWhen drag ends
onSnap(e) => voidWhen snapping
onSettle(e) => voidWhen settled (spring complete)

Methods

MethodDescription
.enable()Enable dragging
.disable()Disable dragging
.setX(value)Set X position (translateX)
.setY(value)Set Y position (translateY)
.stop()Stop dragging
.reset()Reset to origin
.revert()Alias for reset

Properties

PropertyTypeDescription
xnumberCurrent X position
ynumberCurrent Y position
isDraggingbooleanWhether currently dragging

Note: The Draggable API is currently a skeleton implementation. The interactive demo above uses pointer events directly to demonstrate the intended UX pattern. 
const drag = Anim233.draggable('#box', {
  container: '#area',
  x: true,
  y: true,
  releaseStiffness: 300,
  releaseDamping: 20,
  onGrab: (e) => { box.style.background = 'var(--accent)'; },
  onRelease: (e) => { box.style.background = ''; },
  onSettle: (e) => { console.log('settled at', e.x, e.y); }
});

Axis Locking

// Horizontal only
Anim233.draggable('#box', { x: true, y: false });

// Vertical only
Anim233.draggable('#box', { x: false, y: true });

// With snap to grid
Anim233.draggable('#box', {
  x: { snap: 50 },
  y: { snap: 50 }
});
