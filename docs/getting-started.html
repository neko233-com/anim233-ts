Getting Started

Install Anim233 and start animating in under a minute.

Installation

npm / yarn

npm install anim233-ts
# or
yarn add anim233-ts

CDN (UMD)

<script src="https://unpkg.com/anim233-ts@latest/dist/index.min.js"></script>
When loaded via CDN, Anim233 is available as a global on window.

ES Module Import

import Anim233 from 'anim233-ts';
// or import specific functions
import { tween, animate, stagger } from 'anim233-ts';

Basic Usage

Tween (DOTween-style)

Live Demo
Anim233.tween('#box')
  .to({ x: 200 }, 500, 'easeOutCubic')
  .to({ y: -100 }, 500, 'easeInOutQuad')
  .play();

Animate (anime.js-style)

Live Demo
Anim233.animate({
  targets: '#box',
  x: 200,
  rotate: 360,
  duration: 1000,
  ease: 'easeOutCubic',
  loop: false
});

Timeline

const tl = Anim233.timeline();
tl.add({ targets: '#a', x: 200, duration: 500 });
tl.add({ targets: '#b', x: 200, duration: 500 }, '-=200');
tl.play();

Module Structure

Anim233 is a single-file, zero-dependency library. Everything is accessible through the Anim233 namespace:

Anim233.animate()      // Core animation
Anim233.tween()        // DOTween-style chain
Anim233.timeline()     // Sequencing
Anim233.animatable()   // Proxy-based reactive wrapper
Anim233.draggable()    // Drag interaction
Anim233.onScroll()     // Scroll observer
Anim233.splitText()    // Text splitting
Anim233.createShaderAnim233() // WebGL shaders
Anim233.stagger()      // Stagger calculator
// + 14 math utilities
