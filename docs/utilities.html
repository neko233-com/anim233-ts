Utilities API Reference

Utilities

14 math and array utilities included with Anim233. All are available via the global Anim233 namespace or as named imports.

Math Utilities

FunctionSignatureDescription
lerp(start, end, t) => numberLinear interpolation between start and end
damp(start, end, smooth, dt) => numberExponential smoothing (damped lerp)
clamp(value, min, max) => numberClamp value to range
snap(value, gridValues) => numberSnap to grid or nearest array value
wrap(value, min, max) => numberWrap value in range (modulo)
mapRange(value, inMin, inMax, outMin, outMax) => numberLinear range mapping
round(value, decimals?) => numberRound to N decimal places
random(min, max) => numberRandom float in range
createSeededRandom(seed) => () => numberDeterministic PRNG (LCG algorithm)
degToRad(deg) => numberDegrees to radians
radToDeg(rad) => numberRadians to degrees

Array Utilities

FunctionSignatureDescription
randomPick(array) => TRandom element from array
shuffle(array) => T[]Fisher-Yates shuffle (returns new array)

Async Utilities

FunctionSignatureDescription
wait(ms) => Promise<void>Promise-based delay

Stagger

FunctionSignatureDescription
stagger(value, options?) => (index, total) => numberStagger value calculator for per-element delays/offsets

Stagger Options

OptionTypeDescription
startnumberStarting value
fromnumber | 'first' | 'last' | 'center' | 'random'Starting index
reversedbooleanReverse order
easeEasingName | EasingFunctionEasing for stagger distribution
grid[number, number]Grid dimensions [rows, cols]
axis'x' | 'y'Grid axis
modifier(val, index) => numberCustom value modifier
totalnumberTotal items (auto-detected from target)

Demos

lerp (Linear Interpolation)

Live Demo
t = 0.00 
Anim233.lerp(0, 100, 0.5)  // 50
Anim233.lerp(0, 100, 0.75) // 75

clamp

Interactive
Value: 50
Clamped [0, 100] = 50 
Anim233.clamp(50, 0, 100)   // 50
Anim233.clamp(-10, 0, 100)  // 0
Anim233.clamp(150, 0, 100)  // 100

mapRange

Interactive
Input [0..1]: 50..150 
Anim233.mapRange(0.5, 0, 1, 50, 150)  // 100
Anim233.mapRange(0, 0, 1, 0, 360)     // 0
Anim233.mapRange(1, 0, 1, 0, 360)     // 360

random

Anim233.random(0, 100)    // e.g. 47.238...
Anim233.randomPick(['a','b','c'])  // e.g. 'b'
Anim233.shuffle([1,2,3,4,5])     // e.g. [3,1,5,2,4]

Seeded Random

const rng = Anim233.createSeededRandom(42);
rng(); // 0.374... (always same with seed 42)
rng(); // 0.731... (deterministic sequence)

wait (async delay)

async function sequence() {
  console.log('start');
  await Anim233.wait(1000);
  console.log('after 1 second');
  await Anim233.wait(500);
  console.log('after 1.5 seconds');
}

Angle Conversions

Anim233.degToRad(180)  // Math.PI
