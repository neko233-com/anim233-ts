Text Effects SplitText

Split text into individual characters for scramble, reveal, and stagger effects.

Usage

const splitter = Anim233.splitText('#heading', {
  chars: true,
  words: false,
  lines: false,
  accessible: true
});

// Each char is now a <span> in the DOM
console.log(splitter.chars); // HTMLElement[]

Options

OptionTypeDefaultDescription
charsbooleanfalseSplit by characters
wordsbooleanfalseSplit by words
linesbooleanfalseSplit by lines
accessiblebooleanfalseAdd accessible markup
includeSpacesbooleanfalseInclude spaces as chars
debugbooleanfalseDebug mode
classstring''CSS class for spans
wrapstring''Wrapper element tag
clonebooleanfalseClone node instead of modifying

Methods

MethodDescription
.addEffect(fn)Apply effect function to each char element
.revert()Restore original text content
.refresh()Re-split text

Properties

PropertyTypeDescription
charsHTMLElement[]Array of character span elements
wordsHTMLElement[]Array of word elements (when enabled)
linesHTMLElement[]Array of line elements (when enabled)

Scramble Reveal Effect

Live Demo — Click to scramble
Hello Anim233
const text = Anim233.splitText('#heading', { chars: true });
const chars = '!@#$%^&*()_+-=[]{}|;:,.<>?/~`';
text.chars.forEach((el, i) => {
  const original = el.textContent;
  let count = 0;
  const interval = setInterval(() => {
    el.textContent = chars[Math.floor(Math.random() * chars.length)];
    count++;
    if (count > 10 + i * 2) {
      el.textContent = original;
      clearInterval(interval);
    }
  }, 40);
});

Wave Effect

Live Demo — Wave animation on chars
Wave Effect
const text = Anim233.splitText('#heading', { chars: true });
const wave = text.chars.map((el, i) => ({
  targets: el,
  y: [0, -20, 0],
  duration: 600,
  delay: i * 50,
  ease: 'easeInOutCubic'
}));
Anim233.timeline()
  .add(wave)
  .play();
