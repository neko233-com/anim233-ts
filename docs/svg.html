SVG Animation

While Anim233 does not have a dedicated SVG module, you can animate SVG elements using the core animate() and tween() APIs. SVG properties like stroke-dashoffset, opacity, transform, and CSS properties are all animatable.

SVG + Animate API

Anim233 treats SVG elements like any other DOM element. Use CSS properties and SVG attributes to drive animations.

Common SVG Animation Patterns

Stroke Dash Animation (Draw Path)

Live Demo — Draw SVG Path

const path = document.querySelector('#my-path');
const length = path.getTotalLength();
path.style.strokeDasharray = length;
path.style.strokeDashoffset = length;

Anim233.animate({
  targets: path,
  strokeDashoffset: 0,
  duration: 2000,
  ease: 'easeInOutCubic'
});

Tips

Use getTotalLength() on SVG paths to calculate stroke-dasharray for draw animations. For transforms, prefer CSS transforms (transform: rotate(...)) over SVG transform attributes for better compatibility with Anim233.
