Tween API DOTween-style

Sequential chain API inspired by DOTween. Create tweens with .to(), .from(), .delay(), and play with .play().

Create a Tween

const myTween = Anim233.tween(element);
// element can be HTMLElement or HTMLElement[]

Chain Methods

MethodParametersDescription
.to(props, duration?, ease?)Record<string,any>, number, stringAnimate TO properties. Default duration: 400ms, ease: 'easeOutCubic'
.from(props, duration?, ease?)Record<string,any>, number, stringAnimate FROM properties (reverse direction)
.fromTo(from, to, duration?, ease?)Record, Record, number, stringAnimate between explicit start/end values
.set(props)Record<string,any>Set properties immediately (no animation)
.delay(ms)numberPrepend delay before first step
.loop(count?)numberLoop N times or infinitely (default: Infinity)
.yoyo(enable?)booleanReverse direction on each loop (default: true)

Playback Methods

MethodDescription
.play()Start the tween
.pause()Pause the tween
.resume()Resume from paused state
.restart()Reset to beginning and play
.reverse()Reverse the step order
.kill()Stop animation and clear all steps

Callbacks

MethodSignatureDescription
.onBegin(cb)() => voidFired when play starts
.onComplete(cb)() => voidFired when all steps finish
.onUpdate(cb)(progress: number) => voidFired each frame (0..1)
.onLoop(cb)() => voidFired on each loop restart

Read-only Properties

PropertyTypeDescription
durationnumberTotal duration in ms (sum of all steps)
currentTimenumberCurrent elapsed time in ms
progressnumberOverall progress 0..1
isPlayingbooleanWhether tween is playing
isPausedbooleanWhether tween is paused
isCompletebooleanWhether tween has completed

Demos

Chain API: to → to → delay → play

Live Demo
Anim233.tween('#box')
  .to({ x: 200 }, 500, 'easeOutCubic')
  .to({ y: -80, rotate: 180 }, 400, 'easeOutBack')
  .delay(300)
  .to({ x: 0, y: 0, rotate: 360, scale: 1.2 }, 600, 'easeInOutQuad')
  .play();

Loop & Yoyo

Live Demo — 3 loops with yoyo
Anim233.tween('#box')
  .to({ x: 200, rotate: 180 }, 500, 'easeInOutCubic')
  .loop(3)
  .yoyo(true)
  .play();

Callbacks

Live Demo — Callbacks
Anim233.tween('#box')
  .to({ x: 200 }, 500, 'easeOutCubic')
  .onBegin(() => log('Animation started'))
  .onUpdate(p => log(`Progress: ${(p*100).toFixed(0)}%`))
  .onComplete(() => log('Animation complete!'))
  .play();
