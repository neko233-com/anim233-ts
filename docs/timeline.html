Timeline API Sequencing

Orchestrate multiple animations with precise timing. Add animations at absolute offsets or relative positions.

Create a Timeline

const tl = Anim233.timeline({
  autoplay: true,
  delay: 0,
  loop: false,
  alternate: false
});

Timeline Options

OptionTypeDefaultDescription
delaynumber0Initial delay in ms
durationnumber0Total duration (auto-calculated if 0)
loopnumber | booleanfalseLoop count or infinite
alternatebooleanfalseReverse on each loop
reversedbooleanfalseStart reversed
autoplaybooleantrueAuto-start
playbackRatenumber1Speed multiplier

Methods

MethodDescription
.add(anim, offset?)Add AnimationOptions, AnimationInstance, or callback at offset
.set(props, offset?)Set properties at offset
.sync(anim, offset?)Sync an AnimationInstance at offset
.call(fn, offset?)Call function at offset
.label(name, offset?)Add named label
.remove(anim)Remove an animation
.play()Start timeline
.pause()Pause timeline
.restart()Restart from beginning
.reverse()Toggle reverse
.resume()Resume if paused
.seek(time)Seek to absolute time
.stretch(duration)Change total duration
.cancel()Stop timeline
.then(cb)Promise-based completion

Properties

PropertyTypeDescription
durationnumberTotal timeline duration
currentTimenumberCurrent position in ms
progressnumber0..1 progress
pausedbooleanPaused state
reversedbooleanReversed state

Overlapping Timeline

Live Demo — Overlapping with offset
const tl = Anim233.timeline();
tl.add({ targets: '#a', x: 150, rotate: 180, duration: 600 });
tl.add({ targets: '#b', x: 150, rotate: -180, duration: 600 }, '-=400');
tl.add({ targets: '#c', x: 150, rotate: 180, duration: 600 }, '-=400');
tl.add({ targets: '#d', x: 150, rotate: -180, duration: 600 }, '-=400');
tl.play();

Timeline with Callbacks

Live Demo — Call function at offset
const tl = Anim233.timeline();
tl.add({ targets: '#box', x: 200, duration: 500 });
tl.call(() => log('Halfway!'), 250);
tl.add({ targets: '#box', y: -100, duration: 500 }, '+=0');
tl.call(() => log('Complete!'));
tl.play();

Promise-based Timeline

const tl = Anim233.timeline({ autoplay: false });
tl.add({ targets: '#a', x: 200, duration: 500 });
tl.add({ targets: '#b', x: 200, duration: 500 }, '-=200');
await tl.then();
console.log('Timeline finished');
