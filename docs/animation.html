Animate API anime.js-style

Declarative, single-call animation via Anim233.animate(). Configure targets, properties, timing, and callbacks in one object.

Usage

Anim233.animate({
  targets: '#box',
  x: 200,
  rotate: 360,
  duration: 1000,
  ease: 'easeOutCubic',
  autoplay: true,
  loop: 3,
  alternate: true,
  onUpdate: (anim) => console.log(anim.progress),
  onComplete: () => console.log('done')
});

Options

OptionTypeDefaultDescription
targetsstring | HTMLElement | HTMLElement[] | NodeList''CSS selector, element, or collection
durationnumber1000Duration in ms
delaynumber | (el,i,l) => number0Delay before start (supports per-element)
easeEasingName | EasingFunction'easeOutCubic'Easing function
loopnumber | boolean1true = infinite, or N times
loopDelaynumber0Delay between loops
alternatebooleanfalseReverse direction on each loop
reversedbooleanfalseStart in reversed direction
autoplaybooleantrueAuto-start after creation
playbackRatenumber1Speed multiplier

Properties (animate values)

Any CSS property can be animated. Use numbers for CSS pixel values, or string values for other CSS properties.

PropertyTypeExample
xnumbertranslates to translateX
ynumbertranslates to translateY
rotatenumberdegrees of rotation
scalenumberscale factor
opacitynumber0..1 opacity value
widthnumberpixel width
heightnumberpixel height

Callbacks

CallbackSignatureWhen
onBegin(anim) => voidAnimation starts
onComplete(anim) => voidAnimation completes
onUpdate(anim) => voidEach frame
onRender(anim) => voidEach render frame
onLoop(anim) => voidEach loop restart
onPause(anim) => voidAnimation paused
onBeforeUpdate(anim) => voidBefore property update

Instance Methods

MethodDescription
.play()Start/resume animation
.pause()Pause animation
.restart()Reset and play
.reverse()Toggle reversed direction
.alternate()Toggle alternate mode
.resume()Resume if paused
.complete()Jump to end
.cancel()Stop without completing
.revert()Remove all inline styles
.reset()Reset to initial state
.seek(time)Seek to absolute time
.stretch(dur)Change duration
.then(cb)Promise-based completion

async function animateInSequence() {
